간트차트 (AUIGantt) 속성 설정 및 생성법에 대하여

AUIGantt 는 간트차트를 생성하고, 속성, 메소드 및 이벤트를 처리할 수 있는 자바스크립트 객체입니다.

단일의 간트차트를 생성하기 위해서는 다음과 같이 AUIGantt.create() 메소드를 사용합니다.

// 실제로 #gantt_wrap 에 간트차트 생성
myGanttID = AUIGantt.create("#gantt_wrap", ganttProps);

AUIGantt.create() 는 간트차트를 하나 생성하여 id가 myDiv 로 정의된 Div 태그를 찾아 간트차트를 표현합니다.

간트차트를 생성하기 위해서는 반드시 간트 그리드의 필드 설정(columnLayout)이 선행적으로 선언되어야 합니다.

간트차트의 이벤트를 핸들링 하기 위해서는 create() 함수 호출 후에 이벤트를 바인딩하여 핸들링하십시오.

var myGanttID = AUIGantt.create("#myDiv", ganttProps);
  
// 간트 그리드 셀 클릭 바인딩 
AUIGantt.bind(myGanttID, "cellClick", myCellClickHandler);
