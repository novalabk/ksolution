AUIGantt Features

  • AUIGantt 는 Active-X 사용 없이 자바스크립트와 HTML, CSS 로 작성된 웹 간트차트 컴포넌트 입니다.
  • AUIGantt 는 jQuery 또는 기타 JS Framework 에 의존적이지 않습니다. jQuery(또는 다른 JS) 와 함께 또는 단독으로 사용 가능합니다.
  • AUIGantt 는 특정 서버 사이드(PHP, JSP, ASP, .NET)에 종속되지 않습니다. 어떤 서버 사이드, WAS 든 적용 가능합니다.
  • AUIGantt 는 드래그 & 드랍으로 쉽게 작업 이동 및 선행관계를 지정할 수 있습니다.
