데이터 바인딩 - 문서 데이터

AUIGantt 에 의해 작성된 문서 포맷 형식을 볼 수 있습니다. 문서 데이터는 기본 데이터를 비롯 서식, 설정 등이 포함된 형태의 데이터입니다.

데이터 바인딩 - JSON 계층

서버 사이드에서 DB 또는 다른 저장 매체에 있는 데이터를 JSON 계층형태로 작성하였을 때 AUIGantt 에서 어떻게 출력하는지를 볼 수 있습니다.

데이터 바인딩 - JSON 일반

서버 사이드에서 DB 또는 다른 저장 매체에 있는 데이터를 JSON 일반형태로 작성하였을 때 AUIGantt 에서 어떻게 출력하는지를 볼 수 있습니다.

데이터 바인딩 - XML 일반

서버 사이드에서 DB 또는 다른 저장 매체에 있는 데이터를 XML 일반형태로 작성하였을 때 AUIGantt 에서 어떻게 출력하는지를 볼 수 있습니다.

편집 가능 | 보기 전용 모드

필요 및 권한에 따라 편집 가능 및 보기 전용 모드로 설정 및 변경이 가능합니다.

시트 칼럼 필드 설정

작업명, 시작날짜, 종료날짜 등 기본적인 칼럼을 제거하거나 작업 비용, 특징 등 임의의 원하는 칼럼을 추가 할 수 있습니다.

간트 초기 화면 설정

최초 간트 차트가 로딩되고 데이터를 삽입했을 때 사용자에게 보여줄 초기 화면을 설정할 수 있습니다.

베이스라인(baseline) 표현

계획된 날짜와 실제 실행은 다를 수 있습니다. 이 다른 정보를 베이스라인(baseline)으로 출력 할 수 있습니다.

시간 단위 및 확대,축소

지원하는 시간 단위와 시간축 확대, 축소, 이동 등을 어떻게 설정하는지 볼 수 있습니다.

엑셀로 저장하기

작성된 프로젝트를 간트 차트 포함하여 엑셀로 저장 할 수 있습니다.

프로젝트 시작 날짜 변경

기존의 프로젝트 문서를 재사용하기 위하여 프로젝트 시작 날짜를 변경합니다. 모든 작업이 일괄적으로 해당 날짜에 맞게 변경됩니다.

작업 진행률 업데이트

작업 진행율을 특정 날짜 대비로 몇 퍼센트 진행했는지 일괄적으로 변경 할 수 있습니다.

작업일, 공휴일 설정

토요일 또는 일요일에 작업 할지 여부 또는 임의의 공휴일을 설정 할 수 있습니다.

차트 툴팁 사용자 정의

간트차트 바에 바우스 오버 시 나타나는 툴팁의 내용을 사용자 정의 할 수 있습니다.

정렬, 필터 헤더 메뉴 지정

정렬 및 필터로 작업 내역을 쉽게 분석할 수 있습니다.

간트 스타일 설정

간트 차트의 색상 및 텍스트 필드를 설정 할 수 있습니다.

중요 날짜 차트에 표시(핀업)

간트 차트에 중요 날짜(예: 중간 보고 등), 또는 날짜 영역을 표시 할 수 있습니다.

단축키 설정하기

간트 시트(그리드)에 단축키를 설정하여 원하는 명령어를 실행 할 수 있습니다.

작업 메모(이슈) 설정하기

개별 작업에 메모를 설정할 수 있습니다. 이슈 사항이나 팀원에게 알리고 싶은 메세지를 표시합니다.

다국어(리소스번들)

특정 지역에 맞게 메세지 및 날짜 형식을 변경 할 수 있습니다. 예를 들어 2015-09-09(수) 또는 Wed Sep, 9 2015 처럼 지정할 수 있습니다.

리사이징 하기

간트차트를 특정 이벤트에 따라 또는 브라우저 크기에 따라 리사이징하도록 할 수 있습니다.

메세지 띄우기

간트 차트 위에 메세지를 출력 시킵니다.

이벤트 핸들링

셀 선택, 셀 클릭 등의 이벤트 핸들링을 어떻게 하는지 보여줍니다.

부트스트랩 작업 메뉴 작성

작업과 관련된 메뉴를 부트스트랩(Bootstrap)으로 작성한 예제입니다.

활용 예제

  • 간트 차트 일반
    1. 간트 그리드의 칼럼 레이아웃 변경하기
    2. 스케일링 시 애니메이션 효과 없애기
    3. 특정 날짜 범위 기준으로 축 스케일링 하기
    4. 작업 행 높이, 헤더 높이 조절하기
    5. 스플리터(splitter) 사이즈 조절하기
    6. 외부에서 특정 셀, 행 값 변경하기
    7. 요약작업(branch tasks) 전체 또는 부분적으로 열기/닫기
    8. 특정 행(원하는 행) 얻기
  • 특정 행, 열, 셀 수정/편집 제어하기
    1. 특정 행(rows)만 편집 불가 설정하기(특정 행 수정 하지 못하도록 잠금 설정하기)
    2. 특정 열(columns)만 편집 불가 설정하기(특정 열 수정 하지 못하도록 잠금 설정하기)
    3. 특정 조건으로 수정 허용/불허 결정하기
    4. 특정 조건으로 간트 바(bar) 드래깅 수정 허용/불허 결정하기
